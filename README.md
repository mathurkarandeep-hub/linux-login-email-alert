# Linux Login Email Alert 🚀

This project sends an email alert whenever someone logs into or logs out of a Linux server via SSH.

## 🔧 Features
- Detect SSH login/logout
- Sends real-time email notifications
- Lightweight bash script

## 🛠 Technologies Used
- Bash scripting
- Linux (Ubuntu)
- Mail utilities

## 📦 Setup Instructions

1. Clone the repository:
   git clone https://github.com/mathurkarandeep-hub/linux-login-email-alert

2. Navigate to folder:
   cd linux-login-email-alert

3. Make script executable:
   chmod +x alert.sh

4. Run the script:
   ./alert.sh

## 📌 Use Case
Useful for system administrators who want real-time login monitoring for security purposes.
